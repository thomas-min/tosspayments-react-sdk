# Intro

[Tosspayments JS SDK](https://docs.tosspayments.com/reference/js-sdk)를 react에 사용하기 위해 포팅한 라이브러리 입니다.

## Installation

```bash
# 일반 결제 패키지
yarn add @thomas-min/tosspayments-payment-react-sdk
# 결제위젯 패키지
yarn add @thomas-min/tosspayments-payment-widget-sdk
```
